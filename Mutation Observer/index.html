Notes:

To create Mutation Observer - new MutationObserver(). Mutation observer takes in one single argument - new MutationObserver(entries). This argument is a function that is going to be called every single time Mutation Observer observes some time of mutation in the DOM based on some criteria we define

To tell the Observer what to observe we use observe function .observe() which takes in 2 parameters. 1st, the thing we want to observe, and the 2nd all the options for that.

To stop observing we call .disconect() function and no longer look for DOM changes
